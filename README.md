
# International Football Results Analysis ⚽

Analysis of international football matches (1872–2026) using Python.

 Dataset

* 49,071 matches
* 333 countries
* Source: Kaggle

 Key Insights

* Average goals per match: **2.94**
* Most common total goals: **2**
* Highest scoring match: **Australia 31–0 American Samoa (2001)**

Match Outcomes

* Home wins: **49.0%**
* Away wins: **28.27%**
* Draws: **22.73%**
* Home teams win **1.73×** more → clear home advantage

 Top Winning Countries

1. Brazil — 669
2. England — 623
3. Germany — 595
4. Argentina — 586
5. Sweden — 539

 Visualizations

* Goals distribution histogram
* Match outcomes bar chart
* Top 10 countries by wins

Tech Stack

* Pandas
* NumPy
* Matplotlib
